# Alex | Interactive Terminal CV 2.0

> **Warning:** This is not a regular PDF. Accessing this repository grants you entry into a high-performance, containerized career interface.

## Overview

**Alex-CV** — This isn't just a landing page. It's an interactive terminal emulator written in React that serves as an interface for my professional work..

### System Specs:

  * **Engine:** React 19 + Vite (for lightning-fast boot time).
  * **Theme:** Custom Ubuntu/Dark Hybrid.
  * **Features:** Command history, Tab-completion, Geolocation-aware prompts, and real-time GitHub sync.
  * **Infrastructure:** Deployed via GitHub Actions to GitHub Pages (Serverless).

-----

## Available Commands (Quick Start)

If you are already on [amtrend.github.io](https://amtrend.github.io/), try these basic commands:

  * `help` — Список всех доступных модулей.
  * `skills` — Выгрузка стека (Ansible, Docker, K8s, Go, Python).
  * `projects` — Список запущенных в продакшн кейсов.

## Installation & Local Development

If you want to deploy this instance locally:

```bash
# Clone the repository
git clone https://github.com/Amtrend/Amtrend.github.io.git

# We go into the directory
cd Amtrend.github.io

# Install dependencies
npm install

# Run in developer mode
npm run dev
```

## DevOps Pipeline

The project follows CI/CD guidelines. Every change goes through an automated pipeline.:

1.  **Code Push:** Commit to a branch `master`.
2.  **GitHub Action:** Trigger workflow `deploy.yml`.
3.  **Build:** Building an optimized bundle.
4.  **Deploy:** Automatic push to the thread `gh-pages`.
